# MINSAL Invetory App

## Features:

- State: Redux-Toolkit.
- CSS style: Bootstrap
- Api call: axios
- Server: json-server
- Routes: react-router-dom@6

### Intall packages

```
yarn
```

### Runing app

```
yarn dev (for app)
and
yarn start:server (for json-server)
```
